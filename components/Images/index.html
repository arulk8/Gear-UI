Responsive Images

Image helps to understand the website more easily. Images helps to build more pleasing user experience

Images with different sizes

Images with different sizes are available with class .img .img-sm, .img-md and .img-lg for small, medium and large image 

          <img
            src="../../assets/image/plans.jpg"
            class="img img-sm"
            alt="image with laptop"
          />
          <img
            src="../../assets/image/plans.jpg"
            class="img img-md"
            alt="image with laptop"
          />

Fluid images

Images can be made responsive .img-fluid class 

          <img
            src="../../assets/image/plans.jpg"
            class="img img-fluid"
            alt="image with laptop"
          />

Round images

To make Image rounded add .img__round--sm, .img__round--md, .img__round--lg .img-fluid class 

          <img
            src="../../assets/image/plans.jpg"
            class="img img__round--sm"
            alt="image with laptop"
          />
          <img
            src="../../assets/image/plans.jpg"
            class="img img__round--md"
            alt="image with laptop"
          />
          <img
            src="../../assets/image/plans.jpg"
            class="img img__round--lg"
            alt="image with laptop"
          />
