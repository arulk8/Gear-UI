Badges

Badges are used to show additional details to any content. They are used to show the notification count.

Badge on Icons

An icon can have badge on it. example showing count on wishlist in e-commerce app

3
3
3 
          <div class="badge">
            <i class="fas fa-shopping-cart"></i>
            <span class="badge__count bg__secondary c-white">3</span>
          </div>
          <div class="badge">
            <i class="fas fa-heart"></i>
            <span class="badge__count bg__secondary c-white">3</span>
          </div>
