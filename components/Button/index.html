Button

Buttons are components user can click to perform an action and interact with the website.

Primary Button

Primary Button are mainly used for performing success actions. Eg, Submit, Review 

            <button class="btn btn__primary">Submit</button>
            <button class="btn btn__primary--ol">Submit</button>
            <button class="btn btn__success">Submit</button>
            <button class="btn btn__success--ol">Submit</button>

Link Button

Link button are mainly used navigation purpose

Home Docs Navigation Action Get started Cancel 
          <a href="#" class="btn__link btn__link--primary">Home</a>
            <a href="#" class="btn__link btn__link--primary">Docs</a>
            <a href="#" class="btn__link btn__link--success">Navigation</a>
            <a href="#" class="btn__link btn__link--success">Action</a>

Icon Button

Link button are mainly used navigation purpose 

          <button class="btn__icon btn__icon--primary font-bolder">
            <i class="fas fa-address-book"></i>
          </button>
          <button class="btn__text-icon btn__text-icon--primary font-medium">
            <span> Github </span>
            <i class="fab fa-github font-bold"></i>
          </button>

Floating Button

Floating action buttons are used to perform primary or common actions on the screen. Circular in shape, they remain fixed on the bottom left corner of the screen and on top of all content. 

          <a href="#" class="btn__float btn__float--primary box-shadow">
            <i class="fas fa-plus"></i>
          </a>
          <a href="#" class="btn__float btn__float--success box-shadow">
            <i class="fas fa-camera"></i>
          </a>
          <a href="#" class="btn__float btn__float--error box-shadow">
            <i class="fas fa-comment-dots"></i>
          </a>
