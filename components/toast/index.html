Toast

Toasts are basically disappearing notification messages that appear on the screen to inform the user about something.

Basic toast

Add .toast__top or.toast__bottom to show toast on top or bottom of the page

Error occured. 
          <div class="toast toast__bottom hide" id="toast">
            <span>Error occured.</span>
            <button id="toastClose" class="font-medium toast__close">
              <i class="fas fa-times"></i>
            </button>
          </div>
