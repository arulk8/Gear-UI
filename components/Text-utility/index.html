Heading ranges from h1 to h6

h2 - This is h2 heading

h3 - This is h3 heading

h4 - This is h4 heading

h5 - This is h5 heading

Small text

For using small text apply text-sm class to the element

This is small text
            <div class="text-sm">This is small text</div>

Gray text

For using gray text text-gray class to the element

This is gray text
            <div class="text-gray">This is gray text</div>

Center text

For center text center-text class to the element
