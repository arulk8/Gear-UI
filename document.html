Getting Started

To get started with Gear ui we need to link it's css inside head of your html file.

Importing in HTML:

           <link rel="stylesheet" href="https://gearui.netlify.app/library.css" />

Importing in CSS:

          @import url("https://gearui.netlify.app/library.css");
